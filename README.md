# San Diego Air Quality Power BI Dashboard

## Overview

Interactive Power BI dashboard analyzing air-quality conditions, pollutant measurements, geographic variation, and trends across San Diego-area monitoring sites.

## Business Question

How can air-quality data be organized into an interactive dashboard that helps users compare pollutant levels across locations, identify trends over time, and better understand changing environmental conditions?

## Dashboard Features

* Interactive map of monitoring locations
* Pollutant and site-level comparisons
* KPI metrics for air-quality conditions
* Trend analysis over time
* Filters for exploring locations and pollutants
* Data definitions and methodology notes

## Tools Used

* Power BI
* Data visualization
* Geographic analysis
* Trend analysis
* Interactive dashboard design

## Key Skills Demonstrated

* Designing interactive Power BI reports
* Building KPI-driven dashboards
* Communicating data through maps, charts, filters, and trend visuals
* Organizing environmental data for decision support
